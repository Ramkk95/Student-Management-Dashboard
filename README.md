# Student-Management-Dashboard
A full-stack web application to manage student records using FastAPI (backend) and Streamlit (frontend) with SQL Server database.
🛠️ Tech Stack

Backend: FastAPI

Frontend: Streamlit

Database: SQL Server

Language: Python

ORM/DB Tool: SQLAlchemy
